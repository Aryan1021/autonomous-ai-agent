# 🔒 Security Policy

Thank you for helping keep **Autonomous AI Agent** secure.

The security of this project is a top priority. If you discover a security vulnerability, please report it responsibly by following the guidelines below.

---

# 📦 Supported Versions

The following table indicates which versions of the project currently receive security updates.

| Version        | Supported |
| -------------- | :-------: |
| Latest         |    ✅     |
| Older Releases |    ❌     |

Only the latest version of the project receives security updates.

---

# 🚨 Reporting a Vulnerability

If you discover a security vulnerability, **please do not create a public GitHub issue**.

Instead, report it privately by contacting the project maintainer with the following information:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested mitigation (if known)
- Proof of concept (if applicable)

---

# 📋 What to Include

To help investigate the issue quickly, please include:

- Operating system
- Python version
- Dependency versions (if relevant)
- Complete reproduction steps
- Error messages or logs
- Screenshots (if applicable)

---

# ⏱️ Response Process

After receiving a security report, the following process will be followed:

1. Confirm receipt of the report.
2. Investigate and validate the vulnerability.
3. Develop and test a fix.
4. Release a security update if required.
5. Publicly disclose the issue after a fix is available (when appropriate).

Every effort will be made to respond to valid security reports as quickly as possible.

---

# 🤝 Responsible Disclosure

Please allow reasonable time to investigate and resolve the issue before publicly disclosing any security vulnerability.

Responsible disclosure helps protect users of the project.

---

# 🔐 Security Best Practices

When using this project:

- Never commit your `.env` file.
- Never expose your `GEMINI_API_KEY`.
- Keep dependencies up to date.
- Use trusted Python packages only.
- Regularly review project dependencies for vulnerabilities.

---

# ❤️ Thank You

Thank you for helping improve the security of **Autonomous AI Agent**.

Your responsible reporting helps keep the project safe for everyone.
